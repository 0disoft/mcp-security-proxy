# @0disoft/mcp-security-proxy-core

IO-free tool classification, policy evaluation, redaction, and audit helpers for MCP Security
Proxy.

```sh
npm install @0disoft/mcp-security-proxy-core
```

The package enforces policy at the MCP protocol boundary. It is not an operating-system sandbox.

See the [public API documentation](https://github.com/0disoft/mcp-security-proxy/blob/main/docs/library/public-api.md).
