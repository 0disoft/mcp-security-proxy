# @0disoft/mcp-security-proxy-mcp-adapter

JSON-RPC envelope and MCP tool-call adapters for MCP Security Proxy.

```sh
npm install @0disoft/mcp-security-proxy-mcp-adapter
```

The adapters normalize protocol messages for policy evaluation without treating tool schemas as
proof that a tool is safe.

See the [public API documentation](https://github.com/0disoft/mcp-security-proxy/blob/main/docs/library/public-api.md).
