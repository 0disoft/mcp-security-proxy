# @0disoft/mcp-security-proxy-contracts

Policy, decision, audit, ops-event, and validation contracts for MCP Security Proxy.

```sh
npm install @0disoft/mcp-security-proxy-contracts
```

This package includes the versioned JSON schemas under `schemas/`. It does not perform filesystem,
network, or subprocess IO.

See the [public API documentation](https://github.com/0disoft/mcp-security-proxy/blob/main/docs/library/public-api.md).
